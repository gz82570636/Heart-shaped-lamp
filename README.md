# Heart-shaped-lamp 心形灯

一个基于 51 单片机的嵌入式小项目 —— 用 32 颗 LED 拼出心形阵列,是工科生把技术变成温柔的入门级作品\~

# 项目简介

这是一个**语音控制的 LED 心形灯**,核心是用 STC89C51 单片机驱动 32 颗 LED 组成心形阵列。既能练手嵌入式硬件设计 + 代码开发,又能当桌面氛围灯,完美诠释 “工科人的浪漫”\~

# 功能特点

1. **心形 LED 阵列**:32 颗直插 LED 排列成心形,支持多种亮灯模式;
2. **硬件简洁易复刻**:基于 51 单片机 + 基础分立元件,电路原理清晰,PCB 布局紧凑;
3. **入门友好**:代码逻辑简单(主要是 LED 扫描驱动 + 串口指令解析),适合嵌入式新手练手。

# 硬件组成(核心元件)

基于 BOM 清单简化:

* 主控:STC89C51 单片机(DIP40 封装);
* 显示:32 颗直插七彩慢闪 LED(组成心形);
* 控制:语音识别模块 + 拨动开关;
* 辅助:300Ω/10kΩ 贴片电阻、10μF 电解电容(复位电路)、12MHz 晶振等;


# 硬件设计

* **原理图**:分 “单片机核心电路(复位 + 晶振)”“语音模块接口”“LED 心形阵列驱动” 3 个模块;
* **PCB 实物布局**:心形 LED 阵列居右,单片机 + 辅助元件居左,电源 / 接口放在边缘,兼顾美观和实用性(PCB 截图见仓库内`pcb.png`​)。

# 软件逻辑

代码基于 Keil 开发(51 单片机编译环境),核心流程:

1. 单片机初始化:配置 GPIO 为输出模式、初始化串口(与语音模块通信);
2. 语音指令解析:语音模块识别到指令后,通过串口给单片机发指令码;
3. LED 驱动:单片机根据指令码,控制 32 颗 LED 的通断时序,实现对应亮灯模式，并且通过七彩自动慢闪led自动变色的特性，我们可以让想先变色的灯先亮，起到一个按顺序变色的效果。

# 使用说明

1. 给 PCB 上电(通过 Type-C 接口接 5V 电源);
2. 等待语音模块初始化完成(指示灯闪烁);
3. 说出指令(如 “点亮心形灯”),即可触发对应亮灯效果;
4. 拨动开关可手动切换 “语音控制 / 手动模式”。
